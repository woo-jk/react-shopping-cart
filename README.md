### 📍 학습 목표

다수의 컴포넌트를 페이지로 구성하고 복잡해진 상태를 관리합니다.

- `데스크탑` 타겟의 웹 앱을 구현합니다.
- 상태 관리를 위해 `Recoil`을 활용합니다.
- `Router`를 활용해 여러 페이지 전환을 고려합니다.
- 배민상회 서비스 참고

<br>
